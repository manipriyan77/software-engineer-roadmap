🎛️ Box Model Controls (box-sizing: border-box)

📏 box-sizing: border-box

  • Width/height include padding and border
  • Total size stays constant as you adjust padding/border
  • Total width = width (includes everything)
  • Content shrinks when padding/border increase

📦 Block Elements

  • Takes 100% width of parent (or specified width)
  • Starts on a new line
  • Respects all box model properties
  • Renders top to bottom

Inline Elements (Display: Inline)

This is some text with Inline Element 1 and more text with Inline Element 2 flowing naturally in the content. Notice how inline elements don't break the flow and Inline Element 3 stays within the text line.

📝 Inline Elements

  • Flows with text (left to right, top to bottom)
  • Ignores width and height properties
  • Ignores vertical margins
  • Padding doesn't affect line height
  • Only horizontal margins work

Anonymous Boxes

Anonymous boxes are automatically created by the browser when text content exists directly inside a block container without being wrapped in an element.

Visual Example:

This text is an anonymous box! (not wrapped in any element)

This paragraph IS wrapped in a <p> element

This is another anonymous box!

🔍 What are Anonymous Boxes?

  • Created automatically by the browser
  • Wrap text nodes that aren't inside any element
  • Cannot be styled directly with CSS
  • Inherit styles from their parent
  • Help maintain proper block formatting context

Code Example:

<div>
  This text creates an anonymous box
  <p>This is a paragraph element</p>
  This text also creates an anonymous box
</div>

In the example above, the text "This text creates an anonymous box" and "This text also creates an anonymous box" are not wrapped in any HTML elements, so the browser automatically creates anonymous boxes around them.

💡 Key Insights

  • Block Formatting Context: Anonymous boxes help maintain proper layout structure
  • Inherited Properties: They inherit all inheritable CSS properties from parent
  • No Direct Styling: You can't target them with CSS selectors
  • Automatic Creation: The browser creates them as needed during rendering
