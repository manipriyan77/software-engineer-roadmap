CSS Box Model Demonstration

Understanding the Box Model

Every element in CSS is represented as a rectangular box with four layers:

1. Content Box (Green)

The innermost layer where your actual content (text, images, etc.) is displayed.

width and height properties control this area.

2. Padding Box (Blue)

Space between the content and the border. Creates breathing room inside the element.

padding: 20px; or padding-top, padding-right, padding-bottom, padding-left

3. Border Box (Orange/Yellow)

The border that wraps around the padding and content.

border: 10px solid #color;

4. Margin Box (Pink)

The outermost layer - space outside the border, separating this element from others.

margin: 20px; or margin-top, margin-right, margin-bottom, margin-left

Box-Sizing Property

The box-sizing property changes how width and height are calculated:

content-box (default)

Content
Width: 200px
+ Padding: 40px (20px × 2)
+ Border: 20px (10px × 2)
Total: 260px

border-box

Content
Width: 200px
(includes padding & border)
Total: 200px
Pro Tip: Many developers use box-sizing: border-box; globally to make sizing more intuitive: 
* {
    box-sizing: border-box;
}
